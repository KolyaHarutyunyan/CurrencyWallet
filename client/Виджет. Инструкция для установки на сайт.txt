Важно!
Виджет не будет работать без установленного сертификата SSL
Виджет использует технологии криптозащиты, доступные только для сайтов, работающих по протоколу HTTPS (использующие сертификат SSL)

1. У вашего сайта должен быть установлен сертификат SSL
2. Распакуйте скаченный архив в папку /tokenwidget у себя на сервере
3. Проверьте, что по указанном адресу загружается виждет https://yousite.com/tokenwidget/
4. В удобном для вас месте, расзместите в коде вашего сайта следующий код
4.1. Для отображения кошелька
	<iframe src="//tokenwidget/#/wallet" width="100%" height="100%"></iframe>
4.2. Для отображения обменика
	<iframe src="//tokenwidget/" width="100%" height="100%"></iframe>

Как установить сертификат SSL?
(краткая инструкция, или ссылка на толковый мануал)
